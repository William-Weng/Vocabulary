# Vocabulary - 我愛背單字

針對身為外語苦手的我，量身訂製的背單字 App。擁有輕快的背景音樂、正確的發音、可愛的動畫，並支援 Deep Link 功能，讓背單字變得輕鬆有趣！

## ✨ 特點

- 🎵 **輕快背景音樂**：讓學習過程更放鬆
- 🔊 **發音功能**：正確聽感幫助記憶
- 🎬 **可愛動畫**：增加學習樂趣
- 🔗 **Deep Link 支援**：快速加入單字、搜尋或更換 App 圖示
- 📱 **Swift 98.6%**：現代化 iOS 開發

## 🔗 Deep Link 功能

| 連結 | 功能 |
|------|------|
| `word://append/<單字>` | 加入新單字 |
| `word://search/<單字>` | 搜尋該單字 |
| `word://icon/<圖示編號>` | 更新 App 圖示 |

## 🎨 語言字型

| 字型 | 出處 | 功能 |
|------|------|------|
| `KleeOne-SemiBold.ttf` | Google Font | 日文字型 |
| `GamjaFlower-Regular.ttf` | Google Font | 韓文字型 |
| `jf-openhuninn-2.0.ttf` | open 粉圓 - justfont | 中文字型 |

## 📦 安裝

1. Clone 這個倉庫：
   ```bash
   git clone https://github.com/William-Weng/Vocabulary.git
   ```

2. 進入專案目錄：
   ```bash
   cd Vocabulary
   ```

3. 用 Xcode 打開專案並建構：
   ```bash
   open Vocabulary.xcodeproj
   ```

4. 在 Xcode 中選擇你的模擬器或裝置，按下 `Cmd + R` 執行

## 🎵 背景音樂

音樂檔案格式：`.mp3` / `.m4a`  
請在 iTunes 上自行加入背景音樂檔案。
